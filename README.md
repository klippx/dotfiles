# dotfiles

My dotfiles and install script in stow structure, inspired by https://github.com/fsrc/stowedots

## Installation

    git clone https://github.com/klippx/dotfiles.git
    cd dotfiles
    ./install-base.sh
    ./setup-osx.sh
